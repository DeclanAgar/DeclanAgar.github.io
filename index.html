Procedural City Generator

For my masters disseration in which I investigated different methods of procedural city generation, I created and evaluated a procedural city generator algorithm using L-Systems, a rewritting system with Unity 2021.3.5 as the game engine of choice. In order to add variance to the generator, the use of multiple rules and randomness to the algorithm was added and to make it more akin to a city, road, tree and house assets were added.

Escape from Sylvana Dungeon

Escape from Sylvana Dungeon is a 2D roguelike using Unity Engine that was my submission for my Game Technologies and Simulations module in my third year of University.

This game uses A* pathfinding designed to work with Unity's Tilemaps, includes a behavioural flocking algorithm for the bees that can be seen flying around in the starting room of the level, incorporates an audio manager system and a hierarchical behavioural state machine for the goblin enemies.

View the Repository

Viking Islands Graphics Showcase

Viking Islands is a graphics showcase using OpenGL and a custom codebase from the University that was my submission for my Advanced Graphics for Games module in my fourth year of University.

In this project I created a landscape using a heightmap from a noise generator, and by using a shader, added detail to the landscape with a sand texture on levels close to and beneath the sea level, snow towards the peaks of the landscape, rocky textures on steep cliff sides and grass elsewhere. The showcase also includes a spider mesh with an idle animation, reflective animated water and a post-processing blurring effect. For efficient rendering, the scene is controlled by a scene graph.

Temple at Uppsala Unity Graphics Showcase

Temple at Uppsala is a graphics showcase using Unity Engine that was my submission for my Graphics for Games module in third year of University.

This projects landscape and its textures was done through using terrain tools, the project includes a scene hierarchy for relative position movement and rotation for child objects to parent objects, a water shader graph for reflective and animated water, a day/night cycle, a weather system based on the cameras position on the Y axis, wind that causes trees to sway and particles to move in the direction of the wind, lens flare, and the implementation of the universal render pipeline made my Unity.

Goat Simulator Inspired Physics Showcase

This was my Physics Showcase submission using the Newcastle University custom codebase for my Advanced Game Technologies and Simulations module in my fourth year of University.

The project has an implementation of newtonian physics, including collision detection, resolution and application of forces. There is also an implementation of A* pathfinding for navigation through a given maze in a text file, and has a simple bridge using constraints and forces between different gameobjects.
